- Notes on worker strategy:
  - Initial commit `31d7ea7` used the default `@monaco-editor/loader` jsdelivr CDN path — functional but leaks a cross-origin fetch under Tauri's `asset://` scheme.
  - Follow-up `1c7b4cf` pins the loader to the bundled npm package via `loader.config({ monaco })` and wires the `editor.worker` through Vite's `?worker` import. All worker loading now stays same-origin. Tauri's `tauri.conf.json` has `security.csp: null`, so no CSP changes were needed in `src-tauri/capabilities/default.json`.
  - Bundle size: 3.87 MB → 7.22 MB (+3.35 MB, Monaco core). Judge R-5 roadmap explicitly budgeted "~2 MB"; we ran slightly over because self-hosting ships Monaco's full basic-languages set. Acceptable tradeoff for offline-safe operation.
